# Minimize Disturbance

Cho mảng $a$ có độ dài $n$.

Ta có thể chọn vị trí $i$ và hoán đổi:

$$
a_i \leftrightarrow a_{n-i+1}
$$

Mục tiêu là giảm số lượng cặp kề nhau bằng nhau:

$$
a_j = a_{j+1}
$$

sau khi thực hiện các phép hoán đổi.

---

# Ý tưởng

Mỗi cặp đối xứng $(a_l, a_r)$ chỉ ảnh hưởng tới:

- $a_{l-1}$
- $a_{r+1}$

nên ta có thể greedy độc lập cho từng cặp.

Dùng hai con trỏ:

- `l` từ trái sang
- `r` từ phải sang

Tại mỗi bước:

- tính số conflict hiện tại
- tính số conflict nếu swap
- nếu swap tốt hơn thì thực hiện swap

---

# Độ phức tạp

- Time: $O(n)$
- Memory: $O(1)$

---

# Code

```cpp
#include <bits/stdc++.h>
using namespace std;
#ifdef uwu
#include "thcte/debug.h"
#else 
#define dbg(...) 22
#endif

using ll = long long;

signed main() {
    ios::sync_with_stdio(false); 
    cin.tie(0); 
    cout.tie(0);

    int t;  
    cin >> t; 

    while (t--) {
        int n; 
        cin >> n; 

        vector<int> a(n);

        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }

        int res = 0, l = 0, r = n - 1;

        while (l < r) {
            int kd = 0, d = 0;

            if (l > 0 && a[l] == a[l - 1])
                kd++;

            if (r < n - 1 && a[r] == a[r + 1])
                kd++;

            if (l > 0 && a[r] == a[l - 1])
                d++;

            if (r < n - 1 && a[l] == a[r + 1])
                d++;

            if (d < kd)
                swap(a[l], a[r]);

            if (l > 0 && a[l] == a[l - 1])
                res++;

            if (r < n - 1 && a[r] == a[r + 1])
                res++;

            if (l + 1 == r && a[l] == a[r])
                res++;

            l++;
            r--;
        }

        if (l == r) {
            if (l > 0 && a[l] == a[l - 1])
                res++;

            if (r < n - 1 && a[r] == a[r + 1])
                res++;
        }

        cout << res << '\n';
    }

    return 0;
}
````
