# Minimize Disturbance

Cho mảng \(a\) có độ dài \(n\).

Ta có thể chọn vị trí \(i\) và hoán đổi:

\[
a_i \leftrightarrow a_{n-i+1}
\]

Mục tiêu là giảm số lượng cặp kề nhau bằng nhau:

\[
a_j = a_{j+1}
\]

sau khi thực hiện các phép hoán đổi.

## Ý tưởng

Nhận xét rằng mỗi cặp đối xứng \((a_l, a_r)\) chỉ ảnh hưởng tới:

- \(a_{l-1}\)
- \(a_{r+1}\)

nên ta không cần xét toàn bộ mảng mỗi lần swap.

Ta dùng hai con trỏ:

- `l` từ trái sang
- `r` từ phải sang

Tại mỗi bước:

- tính số conflict hiện tại
- tính số conflict nếu swap
- nếu swap giúp giảm conflict thì thực hiện swap

Vì mỗi cặp được xử lý độc lập nên thuật toán chạy tuyến tính.

## Độ phức tạp

- Time: \(O(n)\)
- Memory: \(O(1)\)

## Code

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef uwu
#include "thcte/debug.h"
#else
#define dbg(...) 22
#endif

using ll = long long;

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(0); 
    cout.tie(0);

    int t;
    cin >> t;

    while (t--) {
        int n;
        cin >> n;

        vector<int> a(n);

        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }

        int res = 0;
        int l = 0, r = n - 1;

        while (l < r) {
            int cur = 0;
            int swp = 0;

            if (l > 0 && a[l] == a[l - 1])
                cur++;

            if (r < n - 1 && a[r] == a[r + 1])
                cur++;

            if (l > 0 && a[r] == a[l - 1])
                swp++;

            if (r < n - 1 && a[l] == a[r + 1])
                swp++;

            if (swp < cur)
                swap(a[l], a[r]);

            if (l > 0 && a[l] == a[l - 1])
                res++;

            if (r < n - 1 && a[r] == a[r + 1])
                res++;

            if (l + 1 == r && a[l] == a[r])
                res++;

            l++;
            r--;
        }

        if (l == r) {
            if (l > 0 && a[l] == a[l - 1])
                res++;

            if (r < n - 1 && a[r] == a[r + 1])
                res++;
        }

        cout << res << '\n';
    }

    return 0;
}
````
