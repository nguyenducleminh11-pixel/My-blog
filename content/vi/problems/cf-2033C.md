# Minimize Disturbance

![Minimize Disturbance](/2033C.jpg)

Cho mảng $a$ có độ dài $n$. Ta có thể chọn vị trí $i$ và thực hiện phép hoán đổi đối xứng:

$$a_i \leftrightarrow a_{n-i+1}$$

**Mục tiêu:** Giảm tối đa số lượng cặp kề nhau có giá trị bằng nhau sau khi thực hiện các phép hoán đổi.

---

## Ý tưởng giải thuật

Mỗi cặp đối xứng $(a_l, a_r)$ khi hoán đổi chỉ ảnh hưởng đến các phần tử lân cận đã được xét. Ta dùng **Hai con trỏ** duyệt từ ngoài vào.

---

## Mã nguồn C++

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef uwu
#include "thcte/debug.h"
#else 
#define dbg(...)  22
#endif
using ll = long long;

signed main(){
    ios::sync_with_stdio(false); 
    cin.tie(0); cout.tie(0);
    int t; cin >> t; 
    while(t--){
        int n; cin >> n; 
        vector <int> a(n);  
        for (int i = 0; i < n; i++) cin >> a[i];
        int res = 0, l = 0, r = n - 1; 
        while (l < r){
            int kd = 0, d = 0; 
            if (l > 0 and a[l] == a[l-1]) kd ++; 
            if (r < n - 1 and a[r] == a[r+1]) kd ++;
            if (l > 0 and a[r] == a[l-1]) d ++; 
            if (r < n - 1 and a[l] == a[r+1]) d ++;
            if (d < kd) swap(a[l], a[r]);
            if (l > 0 and a[l] == a[l-1]) res ++; 
            if (r < n - 1 and a[r] == a[r+1]) res ++;
            if (l + 1 == r and a[l] == a[r]) res ++;
            l ++; r --; 
        }
        if (l == r){
            if (l > 0 and a[l] == a[l-1]) res ++; 
            if (r < n - 1 and a[r] == a[r+1]) res ++;
        }
        cout << res << '\n';
    }
    return 0;
}
