# Minimize Disturbance

Cho mảng $a$ có độ dài $n$. Ta có thể chọn vị trí $i$ và thực hiện phép hoán đổi đối xứng:

$$a_i \leftrightarrow a_{n-i+1}$$

**Mục tiêu:** Giảm tối đa số lượng cặp kề nhau có giá trị bằng nhau ($a_j = a_{j+1}$) sau khi thực hiện các phép hoán đổi.

---

## Ý tưởng giải thuật

Mỗi cặp đối xứng $(a_l, a_r)$ khi hoán đổi chỉ ảnh hưởng đến các phần tử lân cận đã được xét (nằm ở phía ngoài). Do đó, ta có thể áp dụng chiến thuật **Tham lam (Greedy)** độc lập cho từng cặp từ ngoài vào trong.

Sử dụng kỹ thuật **Hai con trỏ (Two Pointers)**:
- `l` chạy từ đầu mảng ($0$).
- `r` chạy từ cuối mảng ($n-1$).

Tại mỗi bước, ta so sánh:
1. Số lượng xung đột hiện tại (`kd` - không đảo).
2. Số lượng xung đột nếu thực hiện swap $a_l$ và $a_r$ (`d` - đảo).

Nếu việc swap làm giảm số lượng xung đột (`d < kd`), ta tiến hành swap.

---

## Độ phức tạp
- **Thời gian:** $O(n)$ do chỉ duyệt mảng một lần.
- **Bộ nhớ:** $O(n)$ để lưu trữ mảng đầu vào.

---

## Mã nguồn C++

```cpp
#include <bits/stdc++.h>
using namespace std;

/**
 * Giải bài toán Minimize Disturbance bằng Greedy + Two Pointers
 * Mỗi cặp (l, r) đối xứng được tối ưu dựa trên các phần tử lân cận đã chốt.
 */

void solve() {
    int n; 
    if (!(cin >> n)) return;

    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    int res = 0;
    int l = 0, r = n - 1;

    while (l < r) {
        int kd = 0, d = 0;

        // Tính xung đột nếu giữ nguyên
        if (l > 0 && a[l] == a[l - 1]) kd++;
        if (r < n - 1 && a[r] == a[r + 1]) kd++;

        // Tính xung đột nếu thực hiện swap(a[l], a[r])
        if (l > 0 && a[r] == a[l - 1]) d++;
        if (r < n - 1 && a[l] == a[r + 1]) d++;

        // Greedy: Nếu swap tốt hơn thì thực hiện
        if (d < kd) {
            swap(a[l], a[r]);
        }

        // Đếm tổng số xung đột sau khi đã tối ưu cặp (l, r)
        if (l > 0 && a[l] == a[l - 1]) res++;
        if (r < n - 1 && a[r] == a[r + 1]) res++;

        // Case đặc biệt: hai phần tử ở chính giữa mảng kề nhau
        if (l + 1 == r && a[l] == a[r]) {
            res++;
        }

        l++;
        r--;
    }

    // Xử lý phần tử trung tâm nếu mảng có độ dài lẻ
    if (l == r) {
        if (l > 0 && a[l] == a[l - 1]) res++;
        if (r < n - 1 && a[r] == a[r + 1]) res++;
    }

    cout << res << "\n";
}

int main() {
    ios::sync_with_stdio(false); 
    cin.tie(nullptr); 

    int t;  
    if (!(cin >> t)) return 0; 

    while (t--) {
        solve();
    }

    return 0;
}
