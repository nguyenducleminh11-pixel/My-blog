# Minimize Disturbance

Cho mảng $a$ có độ dài $n$.

Ta có thể chọn vị trí $i$ và hoán đổi:

$$
a_i \leftrightarrow a_{n-i+1}
$$

Mục tiêu là giảm số lượng cặp kề nhau bằng nhau:

$$
a_j = a_{j+1}
$$

sau khi thực hiện các phép hoán đổi.

---

# Ý tưởng

Mỗi cặp đối xứng $(a_l, a_r)$ chỉ ảnh hưởng tới:

- $a_{l-1}$
- $a_{r+1}$

nên ta có thể greedy độc lập cho từng cặp.

Dùng hai con trỏ:

- `l` từ trái sang
- `r` từ phải sang

Tại mỗi bước:
- tính số conflict hiện tại
- tính số conflict nếu swap
- nếu swap tốt hơn thì thực hiện swap

---

# Độ phức tạp

- Time: $O(n)$
- Memory: $O(1)$

---

# Code

```cpp
#include <bits/stdc++.h>
using namespace std;

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;

    while (T--) {
        int n;
        cin >> n;

        vector<int> a(n + 1);

        for (int i = 1; i <= n; i++) {
            cin >> a[i];
        }

        int l = 1, r = n;

        while (l < r) {
            int cur = 0;
            int swp = 0;

            if (l > 1) {
                cur += (a[l] == a[l - 1]);
                swp += (a[r] == a[l - 1]);
            }

            if (r < n) {
                cur += (a[r] == a[r + 1]);
                swp += (a[l] == a[r + 1]);
            }

            if (swp < cur) {
                swap(a[l], a[r]);
            }

            l++;
            r--;
        }

        int ans = 0;

        for (int i = 1; i < n; i++) {
            ans += (a[i] == a[i + 1]);
        }

        cout << ans << '\n';
    }

    return 0;
}
```
